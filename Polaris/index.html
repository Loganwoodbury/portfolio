D&D Companion

D&D Companion

Personal Project

Full-Stack application created to help users run a game of Dungeons and Dragons more easily. Utilizes Java, SpringBoot, PostgresSql, React and Javascript, and converts a MongoDB to be used in Java classes and Sql database. Comnects to a third party api to retrieve data on Monsters, Spells, etc. and backend converts that to usable classes and saves the data to a Postgres database. React web app to display information on monsters, a journal that persists data to a database and utilizes html canvas to create a battle map with image upload to help run a game session.
